# Forza Horizon 6 自动脚本 使用说明

## 一、游戏前置配置（必须）

### 1. 共享代码导入
| 类型 | 共享代码 |
|------|----------|
| 刷技术点车辆 | `772778773` |
| 熟练度蓝图地图 | `705399298` |

进入游戏后在「创意作品」中搜索以上代码，分别下载车辆和蓝图。

### 2. 车辆设置
- 将刷技术点的车辆（共享代码 `772778773`）**设置为收藏夹第一位** 把前面的取消收藏
- 脚本买车流程会自动定位到收藏第一位的车辆，位置不对会导致流程跑偏

### 3. 家园设置
- **不要将家设置为「屋敷」**
- 屋敷退出后没法直接进蓝图

### 4. 提前点赞
- 提前对蓝图（`705399298`）和调教（`772778773`）的作者 **各点一次赞**
- 若未点赞，进入蓝图时会弹出点赞提示框，打断脚本的按键序列，不过也没那么重要重新跑就行

### 5. 游戏 UI 设置
- **不要修改游戏默认 UI 布局和按键方式**，脚本依赖固定的菜单位置和按键顺序
- 游戏建议使用**窗口化模式**，底部不要有额外黑边遮挡

- ### 6. 其他注意项
- 刷点数和买车的关键步频时间都参数化了，可以自己调整
- 在跳转导航中，大部分按键、导航时间按照steamdeck时间设计的，启动前务必测试下调试导航，如果连贯不上，调整SEQ_MTG和SEQ_GTM参数
- steamdeck版的脚本功能较win版少，但是足够用，自行摸索安装依赖，不教学

## 二、软件环境

### 安装
- [AutoHotkey v2.0](https://www.autohotkey.com/)（v1.x 不兼容）
- 不想下载直接用我封装好的exe也可以，但是就没法按照配置调参数了（大概率是不需要调的）

### 强烈推荐
- **DisplayFusion**：多显示器管理工具
  - 游戏可后台运行，并防止失去焦点

## 三、热键说明

| 热键 | 功能 | 启动条件 |
|------|------|----------|
| `F5` | 自动大循环（无限） | 在蓝图加载完成的准备页面按下 |
| `F1` | 手动买车+加点 | 在「购买与出售」页面按下，再按停止 |
| `F6` | 调试：蓝图→车库导航 | 任意时刻（调试用） |
| `F7` | 调试：车库→蓝图导航 | 任意时刻（调试用） |
| `F8` | 手动熟练度循环（无限） | 在蓝图加载完成的准备页面按下，再按停止 |
| `F9` | W 循环（持续油门） | 在巨汉赛段开跑后按下，再按停止 |

## 四、F5 自动大循环流程

每个周期自动执行以下四个阶段，无限循环：

```
① 刷熟练度 109 圈
      ↓
② 导航：蓝图 → 车库
      ↓
③ 买车 + 加点（33 次）
      ↓
④ 导航：车库 → 蓝图
      ↓
重复 ①
```

**启动方法：** 进入蓝图，等加载完毕出现「开始竞赛赛事」的准备画面后按 `F5`。

## 五、GUI 窗口操作

- **拖动**：点住窗口顶部标题栏区域（约 35px 高）拖拽即可移动
- **最小化**：点击右上角 `─` 按钮，或从任务栏点击窗口图标
- **关闭脚本**：点击右上角 `✕` 按钮
- **停止任意循环**：再按一次对应热键即可中断

---

## 六、注意事项

- 脚本使用**后台发键**（ControlSend），游戏窗口不需要保持前台焦点
- W 键油门走的是游戏消息队列，如游戏使用 Raw Input 则后台 W 无效（属游戏限制）
- 运行期间**关闭中文输入法**，防止字母键被输入法截获
- 色块检测（购车完成判断、结算画面判断）依赖 PrintWindow，游戏窗口**不能被完全最小化**（可被遮挡，但不能最小化到任务栏）
- 若更换电脑或分辨率，建议先单独跑 F6/F7 导航调试步骤，确认时序正确
